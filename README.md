# Norfolk & Suffolk Guild website

## Stack
- Frontend [Astro](https://astro.build/) and [Vue](https://vuejs.org/) (just for the Photo Gallery)
- Backend [Sanity](https://www.sanity.io/)
- Lives on [Vercel](https://vercel.com/)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

```bash
npx sanity deploy
```
